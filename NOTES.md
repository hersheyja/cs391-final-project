
# BU Challenge Generator — Project Notes
pick a category (optional) → click spin → see a challenge card → it gets added to history.


## User Flow
1. User picks a category (optional) using CategoryFilter
2. User clicks Spin using SpinButton
3. A random challenge appears in ChallengeCard
4. That challenge gets added to HistoryLog

## Categories
- Food
- Campus
- Social
- Academic
